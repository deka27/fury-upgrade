# Sphere Texture

In this tutorial, we will show how to create a sphere with a texture.

<Steps>

### Import necessary modules

```python filename="python" copy
from fury import actor, io, window
from fury.data import fetch_viz_textures, read_viz_textures
```

### Create a scene to start

```python filename="python" copy
scene = window.Scene()
```

### Load an image (png, bmp, jpeg or jpg) using `io.load_image`.

<Callout type="info" emoji="️ℹ️">
  In this example, we will use `read_viz_textures` to access an image of the
  Earth’s surface from the fury Github after using `fetch_viz_textures()` to
  download the available textures.
</Callout>

```python filename="python" copy
fetch_viz_textures()
filename = read_viz_textures('1_earth_8k.jpg')
image = io.load_image(filename)
```

Out:

<Callout emoji="️📤">
  Dataset is already in place. If you want to fetch it again please first remove
  the folder /Users/skoudoro/.fury/textures
</Callout>

### Use `actor.texture_on_sphere` to add a sphere with the texture from your loaded image to the already existing scene.

To add a texture to your scene as visualized on a plane, use `actor.texture` instead.

```python filename="python" copy
scene.add(actor.texture_on_sphere(image))
```

### Record the scene, or set interactive to True if you would like to manipulate your new sphere.

```python filename="python" copy
interactive = False
if interactive:
    window.show(scene, size=(600, 600), reset_camera=False)
window.record(scene, size=(900, 768), out_path='viz_texture.png')
```
